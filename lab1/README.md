# 🤖 TF2 Trading Bot

Telegram-бот для автоматизации торговли предметами Team Fortress 2 на площадке [steam-trader.net](https://steam-trader.net).

## Функционал

- **Автопродажа** — выставляет новые предметы из инвентаря на продажу
- **Андерпрайсинг** — автоматически снижает цену на 1 копейку ниже конкурентов
- **Автопокупка** — покупает предметы из заданного списка по выгодной цене
- **Статистика** — отслеживает продажи, покупки, прибыль
- **Telegram-команды** — управление ботом через чат

## Команды бота

| Команда     | Описание                          |
|-------------|-----------------------------------|
| `/start`    | Приветствие и список команд       |
| `/stats`    | Статистика продаж и покупок       |
| `/selling`  | Предметы, выставленные на продажу |
| `/bought`   | Последние купленные предметы      |
| `/status`   | Текущий статус торгового цикла    |
| `/run`      | Запустить фоновую торговлю        |
| `/stop`     | Остановить торговлю               |
| `/help`     | Справка по командам               |

## Установка и запуск

### 1. Клонировать проект

```bash
git clone <url>
cd tf2-trading-bot
```

### 2. Установить зависимости

```bash
pip install -r requirements.txt
```

### 3. Настроить переменные окружения

Скопировать `.env.example` в `.env` и заполнить:

```bash
cp .env.example .env
```

Заполнить:
- `TG_TOKEN` — токен бота от @BotFather
- `TG_CHAT_ID` — ID чата для уведомлений
- `STEAM_TRADER_API_KEY` — API-ключ steam-trader.net
- `MY_OFFER_PREFIX` — префикс своих лотов (для фильтрации)

### 4. Запустить бота

```bash
python bot.py
```

## Стек технологий

- **Python 3.10+**
- **python-telegram-bot** — работа с Telegram Bot API
- **requests** — HTTP-запросы к API steam-trader.net
- **python-dotenv** — загрузка конфигурации из .env
- **JSON** — хранение статистики
